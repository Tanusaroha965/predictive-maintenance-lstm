# IoT-Based Predictive Maintenance using LSTM Autoencoder

## Overview
This project detects anomalies in industrial IoT sensor data using a BiLSTM Autoencoder and predicts potential failures.

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy, Pandas
- Matplotlib
- Dash

## Dataset
NASA C-MAPSS dataset

## Methodology
- Data preprocessing
- Sliding window technique
- BiLSTM Autoencoder
- Reconstruction error & threshold
- Failure probability using sigmoid

## Results
The model detects anomalies and visualizes system health using graph and dashboard.
